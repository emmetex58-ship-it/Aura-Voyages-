# Aura Voyages 

Aura Voyages is a premium boutique travel agency website designed to inspire travelers to explore hidden gems and embrace slow travel experiences around the world.

The website showcases a luxury travel brand through elegant layouts, immersive imagery, and a refined editorial aesthetic.

---

##  Live Preview

https://emmetex58-ship-it.github.io/Aura-Voyages-/

##  Project Overview

Aura Voyages specializes in:

* Slow travel experiences
* Curated hidden destinations
* Personalized travel planning
* Luxury boutique adventures

The website was developed as a multi-page frontend project using Semantic HTML and Tailwind CSS v4.

---

* Fully responsive design

## ⚙️ Installation & Setup

Clone the repository:

bash
git clone https://github.com/emmetex58-ship-it/Aura-Voyages-.git


Navigate into the project folder:

bash
cd Aura-Voyages-


Install dependencies:

bash
npm install


Run Tailwind CLI:

bash
npx @tailwindcss/cli -i ./CSS/input.css -o ./CSS/output.css --watch


Open the HTML files in your browser or use VS Code Live Server.
